üì¶ Tentang Platform Ini

Platform ini menggabungkan Pterodactyl Panel untuk manajemen server yang powerful dengan aplikasi edukatif yang menyenangkan. Semua berjalan dalam satu infrastruktur Docker yang terkelola dengan baik.

Pterodactyl Panel adalah control panel open-source berbasis web untuk mengelola game server seperti Minecraft, ARK, CS:GO, dan banyak lagi dengan antarmuka yang modern dan responsif.

Tebak Ikan Indonesia adalah game quiz edukatif yang mengajarkan tentang keanekaragaman hayati ikan air tawar Indonesia dari berbagai sungai besar seperti Kapuas, Mahakam, Barito, dan lainnya.

üöÄ Cara Menjalankan

Untuk Pterodactyl Panel:

  1. Edit konfigurasi di docker-compose.yml
  2. Ubah password database dan pengaturan APP_URL
  3. Jalankan: docker-compose up -d
  4. Akses panel melalui browser di port 80/443

Untuk Fish Quiz:

  1. Klik tombol "Main Sekarang" di atas
  2. Atau buka file www/indonesian-fish-quiz.html langsung di browser
  3. Tidak memerlukan instalasi atau konfigurasi tambahan
